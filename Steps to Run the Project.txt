pip install -r /path/to/requirements.txt

Step 4:  Save all the codes in a folder, example  D:\NIDS

Step 4: Launch Visual Studio Code and import all the needed files.

Step 5: Run the Python file and open new terminal.

Step 6: After opening the terminal C:\NIDS\Deploy>

Step 8: Next Navigate to Browser open the localhost.

Step 9: In the localhost enter the username and password.

Step 10: Enter the Network Status Data from the project folder.

Step 11: Next click on the Predict.

Step 12: After that output will be predicted and displayed.
